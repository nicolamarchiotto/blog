Validation of a robot for minimally invasive vertebroplasty

Bogdan Maris, Nicola Marchiotto, Andrea Calanca

2023 12th Annual Conference CRAS, September 2023

Abstract

This paper proposes a robotic system for minimally invasive vertebroplasty, which is attached to the patient using a belt. Error is measured using a prism phantom with holes.
